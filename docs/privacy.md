# Privacy policy - NYTimes Cleaner

**Last updated:** April 2026

## What this extension does

NYTimes Cleaner runs only on **nytimes.com** and **\*.nytimes.com** pages. It changes how videos, carousels, and onsite subscription messages behave in your browser. It operates **entirely on your device** on those sites.

## Data we collect

**We do not collect, store, or transmit personal data** for this extension. There is no analytics, no accounts, and no remote servers operated by the extension author for browsing activity.

The extension uses **host permissions** only so its scripts can run on NYTimes pages, as described in the store listing.

## Third parties

The **New York Times** website may use its own cookies, analytics, and players under **their** policies. This extension does not change how NYTimes handles your data; it only adjusts playback-related behavior in your browser.

## Open source

Source code is available in the project repository. If you have questions, open an issue there.
